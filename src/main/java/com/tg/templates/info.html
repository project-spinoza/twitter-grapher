Graph Paths

  • localhost:8080

    • -By inserting this URL it will show you the default page or information page.

  • localhost:8080/graph

    • -it will show you the sigma graph that uses the twitter hash tags.

Config File

you can use the config file by using -conf config.json

config file has two basic settings:
  • Layout Settings:

    • Layout settings defines what alogrithm graph should use and what should be the shape of graph.

  • App Settings:

    • App settings defines the source of data and the port on which sever listen to the requests.

iFrame/Direct

If you want to directly display the graph you can can insert the URL in web browser.

The other way to display the graph is by using iframe. Make html page using iframe and iframe should requested to url localhost:8080/graph.
